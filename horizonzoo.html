Horizon Zoo

The beginning of my journey as a UI/UX designer. This was the final website we had to build in the Web 1400 Class and I can already see so many changes I would make to the site. While it is not necessarily my most proud project in the world, I think it’s good to remember that we all start somewhere, and this is where that is for me.

I am still very happy looking back at this project. I really enjoy the color pallate that I used and the images that I found really speak to me. The issues that I do have with it really can just be chalked up to being new to the designing world. Strange choices of transparency in the buttons and not having hover effects on buttons. Now that I know better these aren’t choices I would make, but I still love to look back.
