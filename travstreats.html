Trav's Treats

Trav's Treats was an assignment that I did in my first class that really focused on the user experience and design. Not only were we taught and tasked with creating a website, this was also the first class that I was taught about user testing and the intricacies therein.

The site came together through normal processes that every site should go through. User questionnaires, and personas. Then came rough sketching a wire frame, and then creating a more accurate wireframe, followed by a complete working wireframe, all done in Adobe XD. We then tested the working wireframe, before building the site and, yet again, user testing.

Overall, this was the most in depth process that I have undergone to create a site yet and I learned a ton. While this site was completely created in Wix site builder, the site wasn't really the end goal. The goal was to learn more of the process, and in that case I think it was a great show of knowlege. And I think the site turned out really well to top it off.
