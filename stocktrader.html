VueJS Stock Trader Game

For my Web 2630 Class I was tasked with creating a Stock Trading Game using VueJS. As a Web/UX student, I felt very unprepared for this assignment, but I was up for the challenge as I had been learning all about VueJS the whole semester. By the end I had a completely functional game with all of the buttons and functions completely operational.

While this is not visually the best website, it was never meant to be. If tasked, I do believe that a little color here and there could have gone a long way, but that was not part of the assignment so it was not incorporated.
